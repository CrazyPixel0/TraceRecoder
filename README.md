# tz_flask_20230311_route
show route map from gpx file

## how to run 
* export FLASK_APP=app.py
* export FLASK_ENV=development
* flask run

## how to deploy
* static/***.gpx(preprred)
* static/2023.html(created by app.py)
* templates/index.html
* app.py
